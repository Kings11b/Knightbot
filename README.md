## 📝 Commands

### General Commands:

- `.help` or `.menu`
- `.tts <text>`
- `.sticker` or `.s`
- `.owner`

### Admin Commands:

- `.ban @user`
- `.promote @user`
- `.demote @user`
- `.mute <minutes>`
- `.unmute`
- `.delete` or `.del`
- `.kick @user`
- `.warnings @user`
- `.warn @user`
- `.antilink`

### Game Commands:

- `.tictactoe @user`
- `.move <position>`

### Group Management:

- `.tagall`

### Other:

- `.topmembers`
